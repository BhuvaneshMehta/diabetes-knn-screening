# Diabetes risk screening with KNN (DAAI SS26)

KNN-based screening pipeline for the Pima Indians Diabetes dataset, built for the
Digital Health Data Analytics & AI project report.

## Setup

```
pip install pandas numpy scikit-learn matplotlib joblib
```

Download `diabetes.csv` (Pima Indians Diabetes dataset) from Kaggle
(https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database)
and place it in this folder.

## Run

1. `python train_model.py` — cleans the data, plots error rate vs k, runs grid
   search, trains the final model (k=19, Euclidean), prints the test-set metrics
   and saves `knn_model.joblib` + `scaler.joblib`.
2. `python gui_app.py` — opens the desktop screening tool: eight input fields,
   one Predict button, shows the risk flag and the neighbour vote share.

## Files

- `train_model.py` — full pipeline: cleaning, scaling, k selection, evaluation
- `gui_app.py` — Tkinter GUI that loads the saved model
- `error_rate_plot.png` — generated by the training script
